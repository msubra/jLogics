Anonymous CVS Access

This project's CVS repository can be checked out through anonymous using these details:

Method pserver
Server cvs.xstream.codehaus.org
Username anonymous
Password (empty)
CVS Root /scm/xstream
Module xstream 
:pserver:anonymous@cvs.xstream.codehaus.org:/scm/xstream

Example

cvs -d :pserver:anonymous@cvs.xstream.codehaus.org:/scm/xstream login
cvs -z3 -d :pserver:anonymous@cvs.xstream.codehaus.org:/scm/xstream co xstream

Developer CVS Access

For commit access, developers require an SSH account.

Method ext (ssh)
Server cvs.xstream.codehaus.org
CVS Root /scm/xstream
Module xstream 
:ext:USERNAME@cvs.xstream.codehaus.org:/scm/xstream
